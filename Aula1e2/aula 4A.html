Aula 4A: Interruptor da Lâmpada (Alternando Estados Booleanos) O que vamos aprender: Como trabalhar com propriedades lógicas/booleanas (true ou false) para alternar estados e modificar elementos visuais do HTML com base nisso. O que o código faz: Criamos o objeto lampada com a propriedade ligada iniciando em false (desligada). O método alternar usa o operador de negação (!this.ligada) para inverter o estado atual: se for falso vira verdadeiro, se for verdadeiro vira falso. Na interface, usamos uma condicional para exibir o texto "LIGADA" ou "DESLIGADA" de acordo com o estado do objeto. --> 4A - Fazer Junto: Lâmpada
